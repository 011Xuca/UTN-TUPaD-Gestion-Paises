## Descripción
Programa de consola que gestiona un dataset de países (nombre, población,
superficie y continente), permitiendo agregar, actualizar, buscar, filtrar,
ordenar y obtener estadísticas.

## Archivos del proyecto
- `main.py`: punto de entrada, contiene el menú y la interacción con el usuario.
- `gestion.py`: funciones de lógica (filtros, ordenamientos, estadísticas, alta/actualización).
- `archivos.py`: funciones para leer y guardar el archivo CSV.
- `paises.csv`: dataset base con los países.

## Cómo ejecutar
1. Tener Python 3 instalado.
2. Ubicarse en la carpeta del proyecto.
3. Ejecutar:
```
python3 main.py
```
